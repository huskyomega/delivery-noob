# 外送新手村

台灣外送平台新手完整指南靜態網站。

## 檔案結構

```
delivery-noob/
├── index.html          首頁（Hero + 平台比較）
├── foodpanda.html      foodpanda 完整新手指南
├── ubereats.html       Uber Eats 完整新手指南
├── goodbehavior.html   良民證申請教學
├── equipment.html      外送設備推薦
├── css/
│   └── style.css       主樣式（RWD，foodpanda 粉紅色系）
├── js/
│   ├── jquery-3.7.1.min.js  jQuery（本地）
│   └── script.js       互動功能（選單、FAQ、計算機）
└── images/             （目前為空，見下方圖片清單）
```
